Attendance Assistant is an Android Application that helps users particularly students to keep track of their attendance. Most students 
need to maintain a minimum attendance percentage to be eligible to write the final exams. This app helps keep track of the attendance 
percentage of various subjects. 
The main features of the app are:
1) Staff data 
2) Time Table
3) Attendance Log 
4) Attendance Update 
5) Main Page (which displays the attendance of each subject)
